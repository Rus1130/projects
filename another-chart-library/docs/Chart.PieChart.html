Class: PieChart

Chart.PieChart(chartTitle, data, popAmountopt, showPercentagesopt)

new PieChart(chartTitle, data, popAmountopt, showPercentagesopt)

Parameters:
Name Type Attributes Description
chartTitle string the title of the chart
data Array.<object> the data for the pie chart
Properties
Name Type Description
arc number the length of the slice (in percent)
color string the color of the slice
label string the label for the slice
popAmount number <optional>
 how far the slice should pop out when hovered over
showPercentages boolean <optional>
 whether or not to show the percentages of the slices

Example
let chart = new Chart('pie')
    .appendTo('#pie-chart')
    .setData("Favorite Color", [

        { arc: 14, color: 'green', label: 'Green' },
        { arc: 14, color: 'purple', label: 'Purple' },
        { arc: 8, color: 'red', label: "Red"},
        { arc: 7, color: 'black', label: "Black"},
        { arc: 5, color: 'orange', label: 'Orange'},
        { arc: 10, color: 'grey', label: "Other"},
    ], 2, true)
